이번 연습장에서는 프로그램의 흐름을 제어하는 조건문(if, elif, else)을 학습합니다. 이전에 배운 데이터 입력과 수학적 연산자들을 혼합하여 상황에 따라 다른 결과를 내는 코드를 작성해 보세요. 난이도 하(4문제), 중(3문제), 상(3문제) 순서로 배치되어 있습니다.

🎯 핵심 학습 목표

  • ✓ [하] 기본 조건 분기: 단순한 if ~ else 문과 비교 연산자(>=, ==)를 사용하여 상황을 두 가지로 나눕니다.
  • ✓ [중] 다중 조건 분기: elif를 활용하여 3가지 이상의 조건을 처리하고, 논리 연산자(and, or)를 배웁니다.
  • ✓ [상] 복합 논리 및 연산: 수학적 연산(몫, 나머지 등)의 결과값을 활용해 복잡한 조건을 설계하고 형식 지정자로 출력합니다.
