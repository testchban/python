데이터를 바구니에 담는 리스트(List)와 이름표를 붙여 관리하는 딕셔너리(Dictionary)를 학습합니다.
특히 복잡한 데이터를 처리할 때 사용하는 '딕셔너리 리스트'(딕셔너리들이 줄지어 있는 리스트)의 개념을 명확히 익혀보세요.

🎯 핵심 학습 목표

  • ✓ 리스트와 딕셔너리 구분: 순서대로 담는 리스트와 키(Key)로 찾아가는 딕셔너리(`dict[key]`)의 차이를 이해합니다.
  • ✓ 딕셔너리 리스트 구조: 리스트의 각 칸에 딕셔너리 객체가 들어있는 구조를 코드로 구현합니다.
  • ✓ 데이터 필터링: 복합 자료구조에서 특정 키를 기준으로 정보를 추출하여 분석할 수 있습니다.
